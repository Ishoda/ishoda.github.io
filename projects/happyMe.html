Project Overview

HappyMe is a wellness-focused mobile application developed to help users maintain positive daily habits. The application combines habit management, emotional wellness tracking, hydration monitoring and personal progress reporting into one simple platform.

Problem Statement

Many users struggle to maintain healthy routines because they lack a simple and personalized way to track daily habits, monitor emotions and measure their progress. HappyMe addresses this challenge by providing a single platform that supports consistency, mindfulness and self-improvement.

Main Features

Home Dashboard

Provides daily habit overview, motivational quotes, water progress and a 7-day mood summary chart.

Routine Management

Allows users to add, update and track habits. Completed habits are separated from pending tasks.

Mind Tracking

Users can record moods, write reflections and review emotional history.

Fuel Tracker

Tracks daily water intake with reminders and automatic daily reset functionality.

MeTime Profile

Displays statistics, streaks, settings, theme customization and preferences.

Application Workflow

User Onboarding ↓ Select Habits ↓ Daily Tracking ↓ Mood & Water Monitoring ↓ Progress Reports

Technical Implementation

Application Architecture

The application follows a single Activity architecture with multiple Fragments for modular design. 


MainActivity

 ├── Home Fragment
 ├── Routine Fragment
 ├── Mind Fragment
 ├── Fuel Fragment
 └── MeTime Fragment

Data Persistence - SharedPreferences

Local storage is used to maintain user preferences and application progress even after closing the app.

  • User profile information
  • Habit selections
  • Mood history
  • Water intake records
  • Theme preferences
  • Completed habit history

Advanced Features

  • Hydration reminder notifications.
  • PDF progress and mood report generation.
  • Dark mode support.
  • Mood visualization through weekly summaries.
  • Persistent user preferences using SharedPreferences.

UI / UX Design

The application uses a natural green color palette to create a calm and positive wellness experience. The interface focuses on simplicity, readability and motivation through clean layouts and clear progress indicators.
