Movie Revenue Analytics: Impact of Ratings on Box Office Performance

Analyzing whether positive reviews influence audience interest using box office data (2000–2024)

Project Overview

This project investigates whether positive reviews influence audience interest and financial success in the movie industry. Using box office data from 2000–2024, the analysis explores the relationship between movie ratings and worldwide revenue through descriptive analytics, inferential statistics, and predictive modelling.

Problem Statement

Do movies with higher ratings generate higher worldwide revenue?
The project evaluates the statement:
"Positive Reviews Influence Audience Interest"
by measuring the relationship between audience ratings and box office performance.

Dataset

  • Kaggle Box Office Dataset (2000–2024)
  • Approximately 5,000 movie records
  • Release Group | Worldwide Revenue | Domestic Revenue | Foreign Revenue | Year Genre | Rating | Vote Count | Production Country | Language

Technologies

  • Programming Language: R
  • Libraries: ggplot2 | dplyr | dunn.test
  • Statistical Techniques: Exploratory Data Analysis | Shapiro-Wilk Test | Kruskal-Wallis Test | Dunn's Post Hoc Test | Spearman Correlation | Linear Regression

Analytical Approach

Descriptive Analytics

Used to understand movie revenue patterns and rating distributions.

  • Data cleaning and preprocessing
  • Revenue distribution analysis
  • Rating category comparison
  • Exploratory Data Analysis (EDA)

Inferential Analytics

Used statistical tests to validate whether ratings influence revenue.

  • Shapiro-Wilk normality test
  • Kruskal-Wallis test
  • Dunn's post-hoc analysis
  • Spearman correlation

Predictive Analytics

Built a statistical model to estimate revenue based on ratings.

  • Linear Regression Model
  • Train-test split
  • R² evaluation
  • RMSE measurement

Key Findings

  • Higher-rated movies generally achieved higher revenue.
  • Kruskal-Wallis and Dunn's tests confirmed significant differences between rating groups.
  • Spearman correlation showed a positive relationship between ratings and revenue.
  • Linear regression showed ratings influence revenue but alone are not sufficient for accurate prediction.
