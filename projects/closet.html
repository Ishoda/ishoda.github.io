Project Overview

Closet is a fashion shopping application concept created to improve the online shopping experience for Sri Lankan fashion lovers. The application combines multiple clothing brands into a single platform while providing personalized recommendations based on user preferences, measurements and style choices.

Problem & Solution

Problem

Users currently need to switch between different websites, Instagram pages and physical stores to compare clothing collections and prices.

Solution

Closet provides a centralized fashion platform where users can discover brands, compare products and receive personalized suggestions.

Key Features

Unified Store Collection

Browse multiple Sri Lankan fashion brands in one application.

Product Filtering

Filter products based on size, category and price.

AI Designer

Receive recommendations using body measurements and preferences.

Ratings & Reviews

Help users make better purchasing decisions.

Favorites

Save preferred products and brands for future access.
