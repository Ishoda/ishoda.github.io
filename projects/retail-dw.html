Retail Analytics Data Warehouse

Building a Business Intelligence solution to transform retail transaction data into actionable insights using ETL, OLAP and Power BI.

Project Overview

This project focuses on designing and implementing a Retail Analytics Data Warehouse to integrate transactional data from multiple sources. The system enables business users to analyze sales performance, profitability, customer behaviour and store performance through Business Intelligence tools.

Technologies

  • SQL Server
  • SSIS
  • SSAS
  • Power BI
  • Excel

Problem Statement

Retail organizations often store data in different systems such as sales, customers, products and stores. Without a centralized analytical system, extracting business insights becomes difficult. This project solves this problem by creating a Data Warehouse and BI reporting solution.

Solution Architecture

Source Systems

Collected transactional data from multiple sources including sales, customers, products, stores, and Excel/CSV files.

Staging Area

Stored raw data temporarily for validation, cleaning, and preparation before loading into the warehouse.

ETL Process (SSIS)

Extracted, transformed, and loaded data using SSIS packages. Applied data cleaning, transformations, and business rules.

Data Warehouse

Designed a star schema with FactSales and dimension tables to support efficient analytical queries.

SSAS Cube

Developed an OLAP cube to enable multidimensional analysis using operations such as drill-down and slice-and-dice.

Power BI Dashboard

Connected the warehouse with Power BI to visualize sales, profit trends, and store performance insights.

Data Warehouse Design

Star Schema

Designed a star schema with FactSales as the central fact table and multiple dimension tables for analytical processing.

Fact Table

Measures: Sales, Quantity, Discount, Profit.

Dimension Tables

- DimCustomer
- DimProduct
- DimStore
- DimDate

ETL Development

Extract

Collected data from SQL databases, CSV files and Excel sources.

Transform

Performed cleaning, data validation, duplicate removal, and format conversion.

Load

Loaded transformed data into warehouse tables using SSIS packages.

Business Intelligence Analysis

SSAS Cube

Created OLAP cubes to perform multidimensional analysis using drill-down, roll-up, slice and dice operations.

Power BI Dashboard

Created interactive dashboards to monitor sales trends, profit analysis and store performance.

Project Outcomes

  • Successfully developed a centralized retail data warehouse using a star schema architecture.
  • Automated data extraction, transformation, and loading processes using SSIS.
  • Enabled multidimensional business analysis through SSAS OLAP cubes.
  • Created interactive Power BI dashboards to analyze sales trends, profitability, and store performance.
  • Gained practical experience in Business Intelligence workflows, data modelling, ETL development, and data-driven decision-making.

Challenges & Learning

Data Integration

Integrated data from multiple sources including CSV files, Excel sheets, and SQL databases into a unified data warehouse environment.

Data Quality Management

Improved data accuracy by handling missing values, duplicate records, and inconsistent data formats during the ETL process.

Data Warehouse Design

Applied dimensional modelling concepts by designing a star schema with fact and dimension tables for efficient analytical processing.

Business Intelligence Development

Learned how OLAP cubes and Power BI dashboards can transform complex datasets into meaningful business insights.
