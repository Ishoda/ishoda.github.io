Closet — Fashion Shopping Platform

A modern mobile application concept designed to simplify fashion shopping by bringing Sri Lankan clothing brands into one platform with personalized recommendations.

Project Information

  • Module: IT2010 - Mobile Application Development
  • Project Type: Mobile UI/UX Design
  • Platform: Android Application Concept
  • Tools: Figma + Android XML

Project Overview

Closet is a mobile application designed to simplify the way Sri Lankan fashion lovers shop for clothes. Shoppers often waste time switching between websites, Instagram pages and physical stores to compare styles, prices, and collections.
Closet solves this frustration by bringing all popular Sri Lankan clothing brands into a single platform, while offering a personalized shopping experience with smart recommendations based on user preferences and body measurements.

Problem Statement

Sri Lankan shoppers currently lack a centralized, trustworthy and personalized fashion app. Users are forced to jump between multiple platforms, making the shopping process time-consuming and overwhelming.

Solution

Closet provides a unified platform that combines multiple Sri Lankan fashion brands, smart filtering, AI-powered recommendations and a seamless user experience to make fashion shopping convenient and enjoyable.

Key Features

Unified Store Collection

Browse all Sri Lankan fashion brands in one app with ratings, reviews, and store details.

Product Filtering

Filter by size, category, price range, and user preferences.

AI Designer

Enter body measurements, height, and preferred colors to get personalized clothing recommendations.

Ratings & Reviews

Help users make better purchasing decisions through authentic community feedback.

Favorites & Wishlist

Save preferred products and brands for future access and easy reordering.

Design System

  • Tools: Figma | Android XML
  • Design Approach: Material Design | 60-30-10 Rule
  • UI Components: ConstraintLayout | LinearLayout | Reusable Components

Color Palette

The app follows the 60-30-10 rule (60% Black, 30% White, 10% Gold)
This combination creates a premium, elegant, and modern look while ensuring excellent readability and product focus.

UI Implementation

  • Designed using Figma for high-fidelity prototypes and interaction flows.
  • Implemented key screens using Android XML (ConstraintLayout, LinearLayout, etc.).
  • Focused on clean visual hierarchy, consistent spacing, and intuitive navigation.
  • Included onboarding flow, login, home, shop portal, favorites, and profile screens.
